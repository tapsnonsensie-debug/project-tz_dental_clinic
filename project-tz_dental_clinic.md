# project-tz.md — Белая Линия

## 1. Паспорт проекта

- **Компания:** Стоматология «Белая Линия»
- **Ниша:** Частная стоматологическая клиника
- **Город/регион:** Казань
- **Описание:** Терапия, имплантация, ортодонтия, профгигиена, диагностика
- **Цель сайта:** Запись на консультацию / приём, сбор лидов на первичную диагностику
- **Главное действие:** Записаться на приём
- **Вторичное действие:** Задать вопрос / обратный звонок

**Целевая аудитория:**
- Взрослые пациенты 24–55 лет
- Родители, выбирающие клинику для семейного обслуживания
- Важны безопасность, комфорт, честные цены и квалификация врачей

## 3. Дизайн-направление

- **Визуальный тон:** Чистый медицинский premium-light. Стерильность, безопасность, забота, современное оборудование, нестрашная стоматология
- **Цветовая палитра:** Белый, мягкий серо-голубой, акцент мятный или глубокий синий
- **Типографика:** Современный гротеск, заголовки уверенные, межстрочность 140–160%, масштаб через `clamp()`
- **Логотип:** `[logo]` — минималистичный знак + wordmark. Без буквальных зубов, улыбок и клипарт-медицинской символики
- **Фото:** `[16:9 image]` — реальные фото врачей, кабинетов, оборудования, зоны ожидания. Лица спокойные, свет чистый
- **Иконки:** Простые линейные для услуг, преимуществ и шагов
- **Запреты:** Без теней-перегруза, градиентов, стекломорфизма, переизбытка медицинской драматичности

---

## 4. Состав секций (порядок)

1. Header — логотип, навигация, телефон `[phone]`, кнопка «Записаться»
2. Hero — оффер, запись на приём, доверительные маркеры
3. Популярные услуги — терапия, имплантация, чистка, брекеты/элайнеры
4. Почему выбирают клинику — оборудование, врачи, стерильность, гарантии, комфорт
5. Врачи — карточки специалистов с фото
6. Отзывы пациентов — карточки или слайдер
7. Фото клиники — мини-галерея
8. Этапы записи / лечения
9. FAQ — боль, противопоказания, уход, стоимость
10. Контакты + часы работы + Яндекс.Карта
11. Footer — legal-ссылки, реквизиты-заглушки, кнопка «Наверх»

**Desktop-логика:**
- Hero: оффер + правая колонка с карточкой быстрой записи
- Услуги — сетка 2×2 или 3×2
- Врачи — карточки с фото, специализацией, опытом
- Галерея — аккуратный слайдер или грид
- Контакты — карта + информация о парковке

**Квиз:** нет — приоритет на popup запись на приём

**Спецкомпоненты:**
- Popup запись на приём
- Форма обратного звонка
- Слайдер отзывов
- Слайдер врачей
- Галерея клиники

**Правила адаптации:**
- Карточка быстрой записи уходит под оффер на мобиле
- Врачи и отзывы — удобный мобильный слайдер
- Не перегружать мобильный экран крупными фото без смысла
